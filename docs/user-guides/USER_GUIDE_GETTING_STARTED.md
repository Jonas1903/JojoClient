# Getting Started - First Steps Guide

## Welcome to JojoClient!

JojoClient is a Minecraft launcher that lets you easily manage profiles, install mods, and play with friends. Here's everything you need to know to get started.

## What is JojoClient?

JojoClient is software that:
1. Lets you log into your Minecraft account
2. Manages Minecraft installations
3. Automatically downloads game files
4. Installs and manages mods from Modrinth
5. Launches Minecraft with your settings

It replaces the official Minecraft launcher with a simpler, more powerful experience.

## First Launch Setup

### Step 1: Install JojoClient

1. Download JojoClient from the official website
2. Run the installer
3. Choose where to install it
4. Wait for installation to complete
5. Launch JojoClient

### Step 2: Choose Your Storage Location

On first launch, JojoClient asks: "Where should I store Minecraft files?"

This is important because:
- Minecraft files, mods, and worlds are stored here
- Takes up 5-50+ GB depending on how many versions/mods you have
- Should be on a drive with plenty of free space

**Recommendation**: Use your largest drive (usually C:) unless it's nearly full.

### Step 3: Log In

1. Click "Log In" button
2. A Microsoft login page opens
3. Enter your Microsoft email and password
4. Grant JojoClient permission to access your Minecraft account
5. You're logged in!

**Your password is NEVER stored** - only a secure token that proves you're logged in.

### Step 4: Create Your First Profile

1. Go to the **Profiles** tab
2. Click **Create Profile**
3. Enter a name (like "My First World")
4. Choose Minecraft version (latest recommended)
5. Click Create

JojoClient will suggest good Fabric versions automatically.

### Step 5: Create an Installation

1. Your new profile appears in the list
2. Click on it to select it
3. Click **Create Installation**
4. Give it a name (like "Survival World 1")
5. Wait for files to download (5-10 minutes for first time)

### Step 6: Play!

1. Select your installation
2. Click **Play**
3. Minecraft launches
4. Log in with your account (already logged in if you're using the launcher)
5. Enjoy!

## Basic Workflow

### Playing Normally

```
1. Open JojoClient
2. Select an installation
3. Click Play
4. Play Minecraft
5. Exit Minecraft
6. Settings automatically save
```

### Adding Mods

```
1. Select a profile
2. Go to Mods tab
3. Search for a mod
4. Click "Add to Profile"
5. Create new installation (or update existing)
6. Click Play
```

### Switching Between Profiles

```
1. Go to Profiles tab
2. Click different profile
3. Create or select an installation
4. Play with new profile
```

## Key Concepts

### Profiles = Setups

Think of profiles as different "game setups":
- Profile 1: "Vanilla Survival" (no mods)
- Profile 2: "Creative Building" (texture packs)
- Profile 3: "Modpack Quest" (50 mods)

Each profile has its own mods, settings, and versions.

### Installations = Game Instances

Installations are the actual games you play:
- Multiple installations can come from one profile
- Each has its own world
- Each has its own progress

### Mods = Add-ons

Mods are programs that modify Minecraft:
- Add new items, blocks, creatures
- Change gameplay
- Improve performance
- Add quality-of-life features

### Versions = Game Versions

Minecraft releases new versions regularly:
- 1.21.4 (newest)
- 1.20.4 (previous)
- Earlier versions available too

Mods only work on specific versions.

## Main Features

### Play Tab

Your current installation and the Play button. This is where you start the game.

### Profiles Tab

Manage your profiles:
- Create new profiles
- Add mods to profiles
- Delete profiles
- Duplicate profiles

### Mods Tab

- Search for mods on Modrinth
- Add mods to your current profile
- View and manage installed mods
- Enable/disable mods

### Settings Tab

Configure:
- Your account
- Storage location
- Memory for the game
- Window size
- Java settings

## Tips for Beginners

1. **Start with vanilla** - Create a vanilla profile first to make sure everything works
2. **Test mods separately** - Add a few mods at a time to find conflicts
3. **Use latest version** - Latest Minecraft usually has the most mods
4. **Keep Java updated** - The game needs modern Java to run well
5. **Backup important profiles** - Export profiles you spend time setting up

## Common Issues

| Problem | Solution |
|---------|----------|
| "Java not found" | Download Java 21 from adoptium.net |
| Game won't start | Check that mods are enabled, restart JojoClient |
| Mods not working | Verify they're compatible with your Minecraft version |
| Download is slow | Check internet connection, try again later |
| Not enough storage | Free up space or choose a different drive |

## System Requirements

**Minimum:**
- Windows 10 or later (or Linux/Mac equivalent)

**Recommended:**

- Java 21
- 8 GB RAM or more
- 50+ GB free space
