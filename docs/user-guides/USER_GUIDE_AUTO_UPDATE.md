# Auto-Update - Automatic Launcher Updates Guide

## What It Does

Auto-Update automatically keeps JojoClient updated to the latest version with new features, fixes, and improvements. You don't have to manually download anything - the launcher updates itself.

## How It Works

### Update Check

When you start JojoClient:
1. It checks online if a newer version is available
2. If there is, it shows you a notification
3. You can choose to update now or later

### Automatic Download

When you choose to update:
1. JojoClient downloads the new version in the background
2. It verifies the download is genuine (SHA verification)
3. It prepares the update

### Installation

Once downloaded:
1. You can choose when to restart JojoClient
2. On restart, the new version is installed
3. JojoClient restarts with all your settings intact

### What's Preserved

When updating, everything stays the same:
- Your accounts (still logged in)
- Your profiles (all intact)
- Your settings
- Your mod collection
- Everything exactly as it was

## Update Notifications

### What You'll See

When an update is available:
- A banner appears at the top or in settings
- Shows the new version number
- Shows a brief description of what's new
- Gives you update options

### Types of Updates

| Type | Size | Importance |
|------|------|-----------|
| **Patch** (0.1.2 → 0.1.3) | 10-50 MB | Bug fixes, minor improvements |
| **Minor** (0.1 → 0.2) | 30-100 MB | New features, improvements |
| **Major** (0 → 1.0) | 50+ MB | Significant new features, redesigns |

## Update Safety

### Verification

Every update is verified to be:
- Genuine (from official JojoClient sources)
- Not corrupted (hash checking)
- Safe to install

### Rollback

If an update has problems, you can:
- Report the issue
- Download the previous version manually
- Reinstall the older version

Your data is never lost.

## Update Timing

### When to Update

- **Automatic checks**: Every time you start JojoClient
- **Manual checks**: In Settings, click "Check for Updates"
- **Frequency**: Usually weekly to monthly releases

### When NOT to Update

- Just before a gaming session where you don't want delays
- If you're on a slow internet connection
- If you're running an important modpack and worried about compatibility

(Updates are usually very compatible, but you can wait if unsure.)

## Update Process

### Step-by-Step

1. **Notification** - You see "Update Available: v1.2.0"
2. **Choose** - Click "Update Now" or "Remind Me Later"
3. **Download** - New version downloads in background
4. **Ready** - Notification shows "Update Ready to Install"
5. **Restart** - Click "Restart Now" or it installs next time you open JojoClient
6. **Updated** - JojoClient restarts with new version

Total time: Usually 1-5 minutes depending on file size and internet speed.

### Update While Gaming

If JojoClient updates while you're playing:
- You can finish your game session
- Update installs when you close the game
- Next time you open JojoClient, new version is active

You won't be interrupted while playing.

## Update Contents

Typical updates include:

### Bug Fixes
- Fixing crashes
- Fixing mod download issues
- Fixing compatibility problems

### Improvements
- Faster downloads
- Better error messages
- Improved performance
- Better UI

### New Features
- New ways to organize profiles
- New mod search features
- Better export/import
- Quality of life improvements

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Update won't download | Check internet connection, try again |
| Update seems stuck | Wait 10-15 minutes - large files take time |
| Old version after update | Manually check version in About/Settings |
| Performance after update | Clear cache, try restarting computer |
| Issues after update | Downgrade to previous version if needed |

## Manual Update (If Automatic Fails)

If auto-update isn't working:
1. Visit the official website
2. Download the latest version manually
3. Run the installer
4. Replace the old version

Your settings and profiles are preserved.

## Staying Informed

### Release Notes

Each update has release notes showing:
- What changed
- What was fixed
- What's new

You can read these before updating if you want to know details.

### Update Channel

JojoClient updates to:
- **Stable releases** - Fully tested, no issues known
- **Beta releases** (optional) - Early access to new features

Most users stick with stable releases.

## Internet Connection

### Required for Update Check

- Broadband or mobile internet needed
- Update check happens automatically
- You can disable auto-check if on expensive data plan

### Update Download Size

Typical update: 20-150 MB
- Over WiFi: 30 seconds to 5 minutes
- Over 4G: 1-10 minutes depending on speed
- Over slow connection: 10-30+ minutes

## Disabling Auto-Update

If you want to disable automatic update checks (not recommended):
1. Go to Settings
2. Find "Auto-Update" option
3. Toggle it off

You'll need to manually check for updates then.

## Version Information

You can see your current version:
- In the launcher's About section
- In Settings
- In the titlebar

Compare it to the latest version on the website to see if you're current.

## Update History

JojoClient keeps a history of:
- What version you're running
- When you updated
- What changed in each update

This is useful for troubleshooting if something breaks after an update.

## Best Practices

1. **Keep updated** - New versions have important fixes
2. **Read release notes** - Know what's changing
3. **Update during safe times** - Not right before playing
4. **Backup your profiles** - Export them periodically in case of issues
5. **Report issues** - If an update breaks something, let developers know
