# JojoClient

JojoClient is a Fabric-native Minecraft launcher for Windows. Sync your mods, profiles, and worlds across every version — without the manual folder shuffling.

## Features

- Microsoft / Xbox / Minecraft account login with multi-account support
- Automatic Java runtime install (Adoptium Temurin)
- Profile system — mods, configs, keybinds, and worlds in one portable file
- Modrinth integration with transitive dependency resolution
- SHA-1 verification on every downloaded file
- Auto crash recovery
- One-click profile export/import

---

## Install

Download the latest installer from the [Releases page](https://github.com/Jonas1903/JojoClient/releases) and run `JojoClient-Windows-Setup.exe`.
